# Multi-Source Knowledge Base with Routing

**Submitted by:** Ghala Mohammed Alkahtani

Capstone project (Track C) — a RAG assistant built with LangGraph functional
API that routes questions between two knowledge sources (product docs and
company policies), with human-in-the-loop approval, long-term memory, and
an evaluator-optimizer loop.

See `Multi-Source Knowledge Base with Routing.ipynb` for the full working
notebook, code, outputs, and rubric write-up.
